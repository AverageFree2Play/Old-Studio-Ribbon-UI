# Old-Studio-Ribbon-UI
These are custom Next Gen Studio ribbon UI tabs designed to look like the original ribbon tab prior to the forced next gen studio ui update.

# Installation
1. Locate your CustomRibbonTabs folder. It should be in `%LOCALAPPDATA%\Roblox\{user-id}\CustomRibbonTabs`
2. Download the json files in the CustomRibbonTabs folder from the repository and drop them inside your CustomRibbonTabs folder.
